My Portfolio

Ferramentex

An application with modules for students of a digital marketing course to practice lessons. It’s responsive, built as a Single Page Application (SPA) with PHP, MySQL, Slim Framework, and AngularJS. I collaborated with a designer and product owner to bring the design into code.

Technologies: PHP, MySQL, AngularJS, Slim Framework, Bootstrap, jQuery, Gulp.

PEA BMTE

A public web application designed to raise environmental awareness through an interactive quiz. Built with AngularJS and jQuery, this SPA works efficiently in scenarios with low internet connectivity.

Technologies: AngularJS, jQuery, Gulp.

Live Demo: PEA BMTE Quiz

Seminario do Sul

This application works as an interface between applicants to a Theological College and a third-party software that manages college data. It is a SPA, responsive, and facilitates the subscription process through API calls.

Technologies: AngularJS (frontend), PHP (backend).
